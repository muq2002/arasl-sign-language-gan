The conditioning ladder

Five conditional GANs on the same backbone. What changes between them is not capacity — it's how each one is conditioned and supervised. Each rung adds one signal and climbs toward the real-image ceiling.

A → B → C → F → G metric: GAN-test recognition ceiling: 97.5% best: G · 94.6%
How we measure

Recognition = can a real-trained classifier read the fake?

Every model is scored with the GAN-test: train a CNN classifier on real held-out images, then measure its accuracy on generated images. High recognition means the synthetic hand is unambiguously its intended letter. The same classifier reaches 97.5% on real images — that is the ceiling every generator is chasing.

reference classifier on real held-out images 97.52% The gap between a model's bar and this line is its remaining fidelity debt.
The dataset

Real Arabic Sign Language hands

32 letter classes, grayscale, tightly cropped. The variety in lighting, skin tone, sleeves and hand angle is what makes class-faithful generation hard.

The five models

Each rung, its data flow, and where it lands

A Label-only cGAN, pixel L1 the baseline

The generator sees only the class label and noise. Its only structural pressure is a pixel-L1 term against an unaligned real image of that class — so the minimizer drifts toward the class mean (regress-to-mean), crushing diversity.

adversarial pixel L1 · unaligned → class mean

B A + MediaPipe landmark loss structure-as-loss — doesn't pan out

Adds a landmark MSE from MediaPipe Hands. But detection collapses to ~2% on low-res grayscale signs, so the term is masked out for ~98% of samples — and the target is still unaligned.

C Structure-conditioned cGAN the one that works

Changes the conditioning, not just the loss. A per-image 3-channel structure map (Canny + silhouette + distance transform) feeds the generator and a paired discriminator, and the L1 target is the same image the structure came from — spatial correspondence restored, so diversity recovers.

real→ structure map→ Generator (enc–dec)→ fake 128²→ D (image, structure, label)
adversarial aligned L1 · same image
76.1%
F C + landmark fusion the biggest single jump

Model C's backbone, unchanged, plus a frozen landmark regressor applied to fake and aligned real — supervising all samples (no detection gate). This one term lifts recognition 12 points. But it's a saturating MSE contributing ~0.1% of the loss: an under-driven signal.

real→ structure map→ Generator (enc–dec)→ fake 128²→ D (image, structure, label)
adversarial aligned L1 landmark MSE · all samples
87.2%
G Recognition-optimized fusion this study's best

Keeps F and adds three signals aimed at the residual gap — a recognition loss for class confusability, feature-matching for blur, an L1-upgraded landmark term — plus generator weight EMA. Detailed below.

real→ structure map→ Generator (enc–dec)→ fake 128²→ D (image, structure, label)
adversarial aligned L1 landmark L1 · λ→8 feature-matching ×10 recognition CE · λ→1 generator EMA
94.6%
The conditioning signal

What the structure map actually is

Models C, F and G don't get raw pixels as conditioning — they get a 3-channel structure map computed with OpenCV. It has 100% coverage (no detector to fail) and preserves the hand's shape while discarding texture the generator should synthesize itself.

Model G · the upgrade stack

Four changes that target F's residual 11-point gap

F's gap to the ceiling is two things: texture fidelity (pixel-L1 leaves images slightly blurry) and class confusability (a few letters collapse toward kaaf). G adds one signal per failure mode and amplifies the landmark term.

01 · AC-GAN

Recognition loss

A classifier trained on real images (independent seed, kept separate from the evaluator) scores each fake; cross-entropy pushes class-discriminative images.

02 · pix2pixHD

Feature matching

L1 between the discriminator's mid-level activations of fake vs the aligned real target — sharpens the texture pixel-L1 leaves blurry.

03 · strengthened

Landmark L1

Switch MSE→L1 (persistent gradient), weight 2→8, warmup from epoch 5. Pulls harder on the term that already drove C→F.

04 · Yazıcı 2019

Generator EMA

Export an exponential moving average (0.999) of the generator weights — a smoother, higher quality generator at no training cost.

L_G = adversarial + 5.0 · |fake − real|₁ aligned pixel L1 + λ_lm · |R(fake) − R(real)|₁ landmark (λ_lm → 8) + 10 · Σ |Dⱼ(fake) − Dⱼ(real)|₁ feature matching (j = 16², 8²) + λ_cls · CE(clf(fake), y) recognition (λ_cls → 1)
8 GB trick: the real-side targets (D features, real landmarks) are constants, so they're computed outside the gradient tape and only the fake forward passes are retained — three teacher networks + EMA still fit in 8 GB at batch 32.
