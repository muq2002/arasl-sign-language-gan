Each cube is a feature map: it gets wider (more pixels) and thinner (fewer channels) as we go — the network trades depth for detail.

Role of each component

What every block contributes to building the hand sign.

① Input fusion Dense + Concat

The random noise z decides which variation of the sign to draw; the class label says which letter. The label is embedded to 128-d and concatenated with z.

noise(128) + label_emb(128) → 256

② Dense + Reshape Dense→4×4×512

Projects the 256-d vector into a tiny but very deep 4×4 image with 512 channels — the "seed canvas" that holds compressed global layout.

256 → 4×4×512 (BatchNorm + ReLU)

③–⑤ Transposed convs ConvT stride 2

Each step doubles the resolution (4→8→16→32) and halves the channel count. This is where coarse blobs gradually become fingers and palm edges. BatchNorm + ReLU keep training stable.

8×8×256 → 16×16×128 → 32×32×128

⑥ Self-Attention SAGAN @32²

Convolutions only see local neighborhoods. Attention lets every pixel look at all other pixels, so the thumb and pinky stay consistent with the wrist. Starts disabled (γ=0) and switches on as it helps.

32×32×128 → 32×32×128 (+ γ·attention)

⑦–⑧ Upsample to full size ConvT stride 2

Two more doublings (32→64→128) add fine texture and sharp edges at full resolution, dropping to just 32 channels.

64×64×64 → 128×128×32

⑨ Output head Conv 3×3 + tanh

A final convolution collapses 32 channels to 1 grayscale channel; tanh squashes pixels to [-1, 1] — the same range as the real training images. (Kept in float32 for stability.)

128×128×32 → 128×128×1

Spotlight: how Self-Attention works

The one non-obvious block. The bright point is a query pixel; lines show it gathering information from key pixels across the whole 32×32 map.

For each pixel the layer computes three projections:

TensorMeaning
Q (query)"what am I looking for?"
K (key)"what do I offer?"
V (value)"the content I pass on"

Attention = softmax(Q·Kᵀ)·V, then the result is added back scaled by a learned γ (initialised to 0). So early on the block is a no-op and only contributes once it reduces the loss — a safe way to add global reasoning.
